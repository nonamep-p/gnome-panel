# GNOME Extensions - Aylur's Widgets

A comprehensive collection of GNOME Shell extensions including: Battery Bar, Dash Board, DateMenu Mod, Media Player, Power Menu, Workspace Indicator, Notification Indicator, Modified Quick Settings, Background Clock, Dynamic Panel, Window Headerbar, and Stylish OSD.

## Supported GNOME Versions

- GNOME 45, 46, 47, 48

## Features

- **Battery Bar** - Visual battery status indicator
- **Dash Board** - Customizable dashboard with multiple widgets
- **DateMenu Mod** - Enhanced date menu with system info
- **Media Player** - Media playback controls
- **Power Menu** - Quick access to power options
- **Workspace Indicator** - Workspace switcher
- **Notification Indicator** - Notification center
- **Quick Settings Tweaks** - QS modifications
- **Background Clock** - Desktop clock widget
- **Dynamic Panel** - Panel customization
- **Window Headerbar** - Window header modifications
- **Stylish OSD** - On-screen display styling

## Installation

Clone the repository and copy the extension folder to your GNOME extensions directory:

```bash
cp -r widgets@aylur ~/.local/share/gnome-shell/extensions/
```

Then enable it in GNOME Settings or using the Extensions app.

## Configuration

Each widget can be individually enabled/disabled in the extension preferences. Access preferences through GNOME Settings > Extensions > Aylur's Widgets.
